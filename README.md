# Curso de HTML - Exercícios

Este repositório contém os arquivos de exercícios do curso de HTML oferecido pela dio.me, ministrado por Diogo Mainardes.

## Descrição do Curso

O curso abrange os fundamentos do HTML (Hypertext Markup Language), a linguagem de marcação padrão para a criação e estruturação de páginas web.

## Como Contribuir

Se você encontrar erros nos exercícios ou tiver sugestões para melhorar este repositório, sinta-se à vontade para contribuir. Basta seguir estes passos:

1. Faça um fork deste repositório.
2. Crie uma branch para suas alterações: `git checkout -b feature/nova-feature`.
3. Faça as alterações desejadas e comite-as: `git commit -m 'Adiciona nova feature'`.
4. Envie as alterações para a sua branch: `git push origin feature/nova-feature`.
5. Abra um pull request indicando as mudanças propostas.
