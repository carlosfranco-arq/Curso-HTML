Estrutura Básica do HTML

O HTML possui uma estrutura hierárquica básica composta por:

  • <!DOCTYPE html>: Declara a versão do HTML.
  • <html>: Contém todo o conteúdo da página.
  • <head>: Metadados da página.
  • <title>: Define o título da página.
  • <body>: Contém o conteúdo visível.

Falando sobre Tags

No HTML, as tags são elementos que definem e estruturam o conteúdo de uma página. Cada tag é composta por um nome entre colchetes angulares, como <tag>.

Principais Tags:

  • <h1> a <h6>: Cabeçalhos de diferentes níveis, sendo <h1> o mais importante e <h6> o menos importante.
  • <p>: Define um parágrafo de texto.
  • <strong>: Torna o texto forte, geralmente exibido em negrito.
  • <em>: Enfatiza o texto, geralmente exibido em itálico.
  • <i>: Enfatiza o texto em itálico.
  • <a>: Cria um link, com o atributo href especificando o URL de destino.
  • <img>: Incorpora uma imagem, com o atributo src definindo o caminho da imagem.
  • <ul>: Inicia uma lista não ordenada.
  • <ol>: Inicia uma lista ordenada.
  • <li>: Define um item de lista dentro de <ul> ou <ol>.

Exemplos:

    <h1>Título Principal</h1>
    <p>Este é um parágrafo de texto.</p>
    <strong>Texto Forte</strong>
    <i>Texto Em Itálico</i>
    <a href="https://www.exemplo.com">Link Exemplo</a>
    <img src="imagem.jpg" alt="Descrição da Imagem">
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>

Atributos Básicos

As tags podem ter atributos que fornecem informações adicionais, por exemplo:

<a href="https://www.exemplo.com">Link Exemplo</a>

Neste caso, href é um atributo da tag <a>.

Textos

No HTML, existem alguns atributos que podem ser aplicados a tags de texto para estilizar ou modificar o comportamento do conteúdo. Aqui estão alguns exemplos:

Atributos de Texto:

  • <strong> e <em>: Essas tags podem ser combinadas para dar ênfase forte a uma parte do texto.
  • <abbr>: Define uma abreviação ou acrônimo, com o atributo title fornecendo uma descrição completa.
  • <sub> e <sup>: Subscrito e sobrescrito, respectivamente.
  • <del> e <ins>: Representam texto deletado e texto inserido, úteis em edições.
  • <mark>: Destaca parte do texto, muitas vezes utilizado para fins de pesquisa.

Exemplos:

    <p>Texto com <strong>ênfase forte</strong> e <em>ênfase</em>.</p>
    <abbr title="World Health Organization">WHO</abbr>
    <p>10<sup>2</sup> representa 100.</p>
    <p>10<sub>2</sub> representa 1/100.</p>
    <del>Texto antigo</del> e <ins>Texto novo</ins>.</p>
    <mark>Texto destacado</mark>

Esses são apenas alguns exemplos, e a combinação desses atributos pode ser usada para criar uma ampla variedade de estilos e formatações de texto.

Listas Ordenadas e Não Ordenadas

No HTML, você pode criar listas ordenadas e não ordenadas para organizar informações de forma estruturada. Aqui estão alguns detalhes sobre esses elementos:

Listas Não Ordenadas (<ul>):

As listas não ordenadas são usadas quando a ordem dos itens não é importante. Cada item é marcado com um ponto (•), mas a apresentação real pode variar dependendo do navegador e do estilo aplicado.

Exemplo:

    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>

Listas Ordenadas (<ol>):

As listas ordenadas são usadas quando a ordem dos itens é significativa. Cada item é numerado sequencialmente por padrão, mas isso pode ser personalizado.

Exemplo:

    <ol>
        <li>Primeiro item</li>
        <li>Segundo item</li>
        <li>Terceiro item</li>
    </ol>

Aninhamento de Listas:

É possível aninhar listas dentro de outras listas para criar uma estrutura hierárquica.

Exemplo:

    <ul>
        <li>Item 1</li>
        <li>Item 2
            <ul>
                <li>Subitem 2.1</li>
                <li>Subitem 2.2</li>
            </ul>
        </li>
        <li>Item 3</li>
    </ul>

Lembre-se de que o correto aninhamento e fechamento das tags são fundamentais para garantir a validação do HTML.

Links

Links são fundamentais para a navegação em páginas web. No HTML, você utiliza a tag <a> para criar links. Aqui estão alguns detalhes sobre o uso de links:

Tag <a>:

A tag <a> (âncora) é usada para criar hiperlinks. O atributo mais importante é href, que define o URL de destino do link.

Exemplo:

    <a href="https://www.exemplo.com">Visitar Exemplo</a>

Atributo target:

O atributo target define onde abrir o link. O valor _blank abre o link em uma nova janela ou guia.

Exemplo:

    <a href="https://www.exemplo.com" target="_blank">Abrir em Nova Janela</a>

Atributo title:

O atributo title fornece informações adicionais sobre o link quando o usuário passa o mouse sobre ele.

Exemplo:

    <a href="https://www.exemplo.com" title="Site de Exemplo">Visitar Exemplo</a>

Links Internos:

Para criar links internos em uma página, utilize o atributo href com o ID da seção desejada.

Exemplo:

    <a href="#sobre">Ir para a Seção Sobre</a>
